# Standards (minimal)

- **Data:** **Accessor** is the contract; **ORM** only behind blackbox if used. SQL, NoSQL, streams — **fit to domain**, not hype.
- **Code:** OOP for core; **pure functions** for injected behavior. **Strong typing** (e.g. Mypy / TS). **Interface-first.**
- **Language:** Python preferred for backend.
- **Policy:** See `../empirestate/STANDARDS.md`, `../core-platform-spec/policies/`.
